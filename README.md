## 关于 v0.8.0

Novel Forge v0.8.0 是全新架构的商业版本，在 v0.7.x 开源版的基础上进行了全面重构和升级。

### 主要方向

- **更稳定的写作流水线** — 重构核心引擎，提升长篇连续写作的稳定性
- **更精准的质量检测** — 升级 Guard 门禁系统与 AI 腔检测精度
- **更好的用户体验** — 桌面端应用 + CLI 双模式
- **商业级支持** — 提供技术支持和持续更新

### 内测说明

- v0.8.0 当前仅限受邀内测用户使用
- 内测期间不收取费用
- 功能可能随时调整，不保证向后兼容
